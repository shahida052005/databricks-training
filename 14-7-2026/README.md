# SQL Assignment

## Assignment Overview
This assignment demonstrates the implementation of SQL queries using the **Employees** table. It covers basic and intermediate SQL concepts such as data retrieval, filtering, grouping, aggregation, sorting, and pattern matching.
## Table Name
**Employees**
## Database Schema
| Column Name | Data Type |
|-------------|-----------|
| emp_id | INT |
| emp_name | VARCHAR(50) |
| department | VARCHAR(50) |
| salary | INT |
| city | VARCHAR(50) |
| experience | INT |

## Topics Covered
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- DISTINCT
- Comparison Operators
- Logical Operators (AND, OR, NOT)
- IN and NOT IN
- BETWEEN
- LIKE Operator

## How to Run
1. Create the `Employees` table.
2. Insert the provided sample data.
3. Execute each SQL query in sequence.
4. Verify the results.
## Learning Outcomes
After completing this assignment, you will be able to:
- Retrieve data using SQL queries.
- Filter records using different conditions.
- Perform grouping and aggregation.
- Apply HAVING clauses.
- Sort and limit query results.
- Use comparison and logical operators.
- Work with IN, BETWEEN, DISTINCT, and LIKE operators.
